## Suggested attack order

1. **`label` on dashboard: settings** — one component fix likely clears 285 nodes (a
   repeated unlabeled control in the edit-evidence rows).
2. **`select-name`** — one shared `<Select>` missing an accessible name, five views.
3. **`color-contrast`** — token-level fix in the shared palette/badge configs will clear
   most of the 536 nodes at once; per-page node counts identify the worst surfaces.
4. **`aria-prohibited-attr`** — swap `aria-label` on plain `Box`es for a role-bearing
   element or a real heading (design-sense §6.1).
5. **`nested-interactive` / `scrollable-region-focusable`** — component-local fixes.
